# Group By

Take the given map of scores and show the people who has the most score ordered ascendant

## Example:
The following input map:

    "Jack" -> 12
    "Jill" -> 15
    "Tom" -> 11
    "Darla" -> 15
    "Nick" -> 15
    "Nancy" -> 11

Should result in:

    {11=[Tom, Nancy], 12=[Jack], 15=[Darla, Nick, Jill]}
